# スクリーンショット撮影ガイド

READMEに載せる画面を撮影して、このフォルダに以下のファイル名で保存してください。
保存後に伝えてもらえれば、メインのREADMEへ埋め込みます。

## 撮影する3枚（`# 05_GITHUB_STRATEGY.md` の最低3枚に対応）

| ファイル名 | 内容 |
|---|---|
| `01-list.png` | タスク一覧画面（タスクが数件ある状態） |
| `02-add.png` | タスク追加フォームに入力した状態 |
| `03-edit.png` | 編集モーダルを開いた状態 |

## 撮影手順

1. バックエンドを起動（別ターミナル）
   ```bash
   cd backend
   .venv/Scripts/activate   # Windows
   uvicorn app.main:app --reload
   ```
2. フロントエンドを起動（別ターミナル）
   ```bash
   cd frontend
   npm run dev
   ```
3. ブラウザで http://localhost:3000 を開く
4. タスクを2〜3件追加してから、上の3画面を撮影
5. `Win + Shift + S`（Windowsの範囲スクショ）などで撮り、このフォルダに保存

## 補足

- 見栄えのため、タスクは「完了済み」「未完了」が混ざった状態にすると良い
- 余裕があれば操作GIF（追加→完了→編集→削除）もあると効果的
